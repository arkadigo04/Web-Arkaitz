⛁ Server Architecture: Infrastructure and local deployment

This website is not hosted on a traditional commercial hosting service. It has been designed, deployed, and secured entirely from scratch to operate from a local data center (Homelab).

Hardware and Storage System

The site operates on a server managed by TrueNAS SCALE. The storage layer is configured with high-redundancy ZFS pools, defining specific Datasets and managing recursive permissions through strict Access Control Lists (ACLs) in Linux.

Containerized Deployment

The web architecture is isolated in containers. A pure Nginx image configured to serve high-performance static content is used. System permission conflicts (UID/GID) between the TrueNAS host and Docker have been resolved through precise volume mapping.

Networking and Ingress Security

The server does not expose management ports to the public network. All incoming traffic is securely routed through reverse tunnels and proxy rules. The transport layer is secured with SSL/TLS encryption, complemented by Cloudflare's DDoS protection and DNS management.
