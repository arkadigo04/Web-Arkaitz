⛁ Arquitectura del Servidor: Infraestructura y despliegue local

Esta página web no está alojada en un servicio de hosting comercial tradicional. Ha sido diseñada, desplegada y securizada íntegramente desde cero para operar desde un centro de datos local (Homelab).

Hardware y Sistema de Almacenamiento

El sitio opera sobre un servidor gestionado mediante TrueNAS SCALE. La capa de almacenamiento está configurada con pools ZFS de alta redundancia, definiendo Datasets específicos y managing recursivos mediante Listas de Control de Acceso (ACLs) estrictas en Linux.

Despliegue Contenerizado

La arquitectura web está aislada en contenedores. Se emplea una imagen pura de Nginx configurada para servir contenido estático de alto rendimiento. Los conflictos de permisos de sistema (UID/GID) entre el anfitrión TrueNAS y Docker se han resuelto mediante el mapeo preciso de volúmenes.

Redes y Seguridad Ingress

El servidor no expone puertos de gestión a la red pública. Todo el tráfico entrante se enruta de forma segura a través de túneles inversos y reglas de proxy. La capa de transporte está asegurada con cifrado SSL/TLS, complementada por la protección DDoS y gestión DNS de Cloudflare.
