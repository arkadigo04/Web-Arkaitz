## 🛠️ Stack Tecnológico del Portfolio

Este portfolio ha sido desarrollado desde cero priorizando el rendimiento, la accesibilidad y un diseño corporativo limpio (Dark Mode / Minimalista).

* **Estructura:** HTML5 Semántico.
* **Estilos:** CSS3 puro (Flexbox, Grid, variables CSS, animaciones nativas).
* **Diseño:** Mobile-First, totalmente responsive.
* **Internacionalización:** Arquitectura bilingüe (Español / Inglés) integrada.
* **SEO & Rendimiento:** Optimizado con etiquetas meta, Open Graph y marcado estructurado Schema.org (JSON-LD) para máxima visibilidad en motores de búsqueda.
